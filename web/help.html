• Bots are written in Lua.
  • API: ShipControl = your program, API: Ship = your ship.
  • Bottom left = (0,0).
  • Angles are in radians. Right/east = 0, up/north = (π/2).
  • World.constants() describe some game rules and physics.
  • You'll probably need the Lua math library.
  • Try these Ship functions:
    x(), y(), speed(), heading(), fireThruster(angle,force), fireLaser(heading), fireTorpedo(heading,distance)
